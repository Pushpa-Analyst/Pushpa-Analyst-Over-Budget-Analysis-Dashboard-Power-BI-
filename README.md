# Pushpa-Analyst-Over-Budget-Analysis-Dashboard-Power-BI-
This project analyzes organizational spending to identify cost centers that are going over budget.  The dashboard helps track:  Budget allocation (Funds)  Actual spending (Expenditures)  Over-budget situations
//Dataset used in this project was provided as part of a DataCamp course and is used for learning purposes.//

**How to Use**
1. Download the .pbix file
2. Open in Microsoft Power BI Desktop
3. Explore filters to analyze different cost centers and time periods
